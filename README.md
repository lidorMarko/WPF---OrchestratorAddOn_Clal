# UiPath Orchestrator WPF Add-On

This project is a WPF desktop application designed to display and manage data retrieved from UiPath Orchestrator databases.  
The application provides a clean desktop interface for viewing operational and automation-related information.

## Features
- Desktop UI built with WPF
- Data retrieval from SQL-based databases
- Presentation of UiPath Orchestrator-related information
- Structured and maintainable C# codebase

## Tech Stack
- C#
- WPF
- SQL
- UiPath Orchestrator

## Application Overview
- Connects to Orchestrator-related data sources
- Displays structured automation data in a desktop interface
- Designed for internal operational and monitoring use

## Purpose
This project was built to practice desktop application development with WPF and to extend UiPath Orchestrator data visibility through a custom internal tool.
